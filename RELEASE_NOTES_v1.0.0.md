# World Cup 2026 Companion v1.0.0

Initial production-ready release.

## Highlights
- Official-source data architecture (env-configured adapters, validation, normalization, caching, attribution, error logging, honest unavailable states)
- Fixtures page with deep filtering, search, timezone modes and spoiler-free support
- Groups & standings with official tiebreakers and qualification highlighting
- Knockout bracket (Round of 32 → Final)
- Team profiles and player profiles from official squad data
- Stadium pages for all 16 venues with travel info and local clocks
- Match details: live timeline, live stats, team comparison, head-to-head, squads, lineups, sourced-only preview, highlights links
- Where-to-watch with country selector and official rights-holder data only
- Browser notifications and a reminders manager
- Calendar export (.ics + Google Calendar; all/favorites/knockout/showpiece bundles)
- Dark mode (system-aware) and spoiler-free mode
- Favorites + personalized "My World Cup" dashboard
- Match prediction poll and fan reactions (local MVP, DB-ready)
- Tournament stats page with official-data leaderboards
- Official news summaries, travel guides, watch party finder
- Digital printable wall chart
- PWA + offline mode with service worker and install prompt
- Multi-language UI (English, Bangla; es/fr/ar scaffolds)
- Ticket & hospitality section (official links only, safety note)
- Admin source monitor with manual refresh and error log
- Vercel deployment config with cron-based data refresh
